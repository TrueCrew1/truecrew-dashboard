# Code of Conduct

This is a private, closed-team repository. This document sets expectations for
anyone interacting with it (issues, pull requests, discussions) — it is not an
invitation to open community participation.

## Expected behavior

- Be direct and professional in issues and pull requests.
- Keep discussion focused on the work — the code, the task, the decision at hand.
- Assume good faith; disagree on substance, not people.

## Not tolerated

- Harassment, personal attacks, or discriminatory language of any kind.
- Spam, unsolicited promotion, or off-topic content in issues or PRs.
- Sharing credentials, secrets, or other sensitive data in issues, PRs, or commits.

## Reporting problems

If you encounter behavior that violates this code, or have any other concern
about activity on this repository, contact **contact@truecrewllc.com**.

Reports will be handled directly by True Crew LLC. There is no public
moderation team or committee — this is a small, closed project, not an open
community.

## Scope

This applies to all interactions on this repository: issues, pull requests,
commit messages, and code review comments.
