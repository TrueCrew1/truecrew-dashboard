## Purpose

request `req-ms-painting-v2-design-standard`. Goal: the deployed product looks TrueCrew by
default — standardized formatting, color, and layout — making it a resellable TrueCrew asset,
with tenant branding (including M&S) layered on as configuration via the Tenant Branding System.
